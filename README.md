This simple plugin can lock on the closest player like an aimbot

Config explaination
```
List Keys: List of keys names that can be used for parameter "Key For Aimlock"
Key For Aimlock: One of the listed keys, it activate the aimlock // Default: "Inspect"
Authorized Player1 (Steam ID64): Steam ID 64 for a player to enable aimlock
Authorized Player2 (Steam ID64): Steam ID 64 for a player to enable aimlock
Can Aim At Teammates Too (If false Only Aims At Enemies): If true, the plugin locks on enemies and teammates too, if false, it only aims at enemies // Default: false
Can Aim At Dead Players Too (If false Only Aims At Alive Players): If true, the plugin locks on alive players and dead player too, if false, it only aims at alive players // Default: false
Interval In Seconds: The inverse of the plugin's frequency, so 1/f which is the period the plugin // Default: 0.015f```
